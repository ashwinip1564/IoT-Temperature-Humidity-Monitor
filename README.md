# IoT Temperature & Humidity Monitor

## Description
This project monitors temperature and humidity using DHT sensor and ESP8266.

## Components Used
- ESP8266 NodeMCU
- DHT11 Sensor
- WiFi Router

## Working
The sensor reads temperature and humidity.
Data is sent to ThingSpeak cloud.
Values are displayed in real-time graphs.

## Output
Temperature and humidity data shown in ThingSpeak dashboard.
